Projects

Recipe with Inventory System

This project is a Recipe Book combined with an inventory system created using Python. This system saves and tracks recipes, manages a list of supplies, and also keeps a record of suppliers for each supply item available in the system.

Generic Solvers

This project integrates three core elements: Quadratic Spline Interpolation (QSI), Polynomial Regression (PR), and a Diet Problem Solver. All components are developed using R Studio and R Shiny to create a graphical user interface (GUI). The Diet Problem Solver incorporates the simplex method for optimization.

Five Minutes of Hell

This project is a game created using Java, JavaFX, and Eclipse, with collaboration from Lorenz Gabriel. The objective of the game is to survive a wave of zombies until help arrives. Players can achieve this by utilizing and purchasing guns from a merchant and acquiring perks that enhance survival capabilities.
